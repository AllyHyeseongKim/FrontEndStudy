# box-shadow

작성자 : 송지은

작성일 : 2015-09-27

css 레퍼런스 설명: 
- box-shadow : 박스 요소의 그림자를 지정하는 속성이다.
- 속성 값 : none, h-shadow v-shadow blur spread color, inset, initial, inherit

`sample code` : 

```html
<!DOCTYPE html>
<html>
	<head>
		<style>
			div {
			    width: 300px;
			    height: 100px;
			    background-color: yellow;
			    box-shadow: 10px 10px 5px #888888;
			}
		</style>
	</head>
	<body>
		<div></div>
	</body>
</html>
```
