css 레퍼런스 설명: 
 - color : 텍스트에 색상을 입힌다.
 - text-align : 텍스트의 위치를 정렬한다.

`sample code` : 
```sh
<!DOCTYPE html>
<html>
<head>
<style>
p {
    color: red; //p테그 색상 
    text-align: center; // p테크 글자 정렬
}
</style>
</head>
<body>
<p>This is a paragraph.</p>

</body>
</html>
```
