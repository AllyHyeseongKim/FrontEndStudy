### 문서 작성 방법

작성자 : 김동일

작성일 : 2015-09-02 17:29

css 레퍼런스 설명: 
- color : 텍스트에 색상을 입힌다.
- text-align : 텍스트의 위치를 정렬한다.

`sample code` : 

```html
<!DOCTYPE html>
<html>
<head>
<style>
p {
    color: red; /*p테그 색상*/ 
    text-align: center;  /*p테크 글자 정렬*/
}
</style>
</head>
<body>
<p>This is a paragraph.</p>

</body>
</html>
```
