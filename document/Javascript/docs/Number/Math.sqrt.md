#Math.sqrt

####링크 : http://www.w3schools.com/jsref/jsref_sqrt.asp (한글번역) 

### 요약(Summary)

SQRT () 메서드는 숫자의 제곱근을 반환합니다.

#### 반환값

수. 경우 x가 음수 인 NaN이 반환된다

### 구문

Math.sqrt(x)

#### 예

```javascript
숫자의 제곱근을 반환합니다 :
Math.sqrt(9);   // [3]를 반환
var a = Math.sqrt(0);   // [0]를 반환
var b = Math.sqrt(1);   // [-1]를 반환
var d = Math.sqrt(64);   // [8]를 반환
var e = Math.sqrt(-9);   // [NaN]를 반환
