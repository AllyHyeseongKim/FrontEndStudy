# Date.getFullYear


getFullyear() 메소드는 현재 시간에 따라 지정된 날짜의 값을 반환합니다.
`UTC(협정 세계 표준시)를 사용하여 연도 값을 가져오려면 getUTCFullYear 메서드를 사용하세요.`
```javascript
var date = new Date("1/1/01");
document.write(date.getFullYear());

//Output: 1901
```
