### 요약(Summary)

자바 스크립트는 숫자 1 종류 만 가지고있다.
숫자는 소수점없이 작성하거나 할 수 있습니다 :


#### 예

```javascript
가장 가까운 정수로 숫자 라운드 :
var x = 3.14;     // [ A number with decimals]를 반환
var y = 34;    // [A number without decimals]를 반환
var x = 123e5;    // [12300000]를 반환
var y = 123e-5;   // [0.00123]를 반환
